## Mechanism
- 원인 모를 이유로 죽음에서부터 다시 깨어남.
- 그녀와 관련된 단서들을 찾아서 구해내는게 이 게임의 목적
- 진행하면서, 충격적인 진실들을 마주하는 플레이어
- 몇몇 이스터 에그, 도움을 원하는 NPC, 통수 치는 NPC 등 다양한 이벤트 많음
